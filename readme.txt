== Description ==

This plugin makes integration in Push-Apps simple and easy. It creates a simple API to register devices.
And sends a push message when a post is published.

Settings:
- Default message
- Push-Sound of the Device
- Send Post-ID or not

View Link for demo-Code:
[iOS](https://github.com/visualartdesign/YAPS-iOS-Sample "iOS Code Example")
